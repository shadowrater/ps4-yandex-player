# 🔑 Нужен новый GitHub токен

Текущий токен не имеет прав на создание репозиториев.

## Создай новый токен:

1. Перейди на https://github.com/settings/tokens
2. Нажми **"Generate new token (classic)"**
3. Назови: `ps4-build`
4. Выбери scopes:
   - ✅ `repo` (полный доступ к репозиториям)
   - ✅ `workflow` (для GitHub Actions)
5. Нажми **"Generate token"**
6. Скопируй токен

## Вставь токен сюда:

```
ghp_ВАШ_ТОКЕН_ЗДЕСЬ
```
